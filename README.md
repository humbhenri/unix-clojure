# unix-clojure

Unix utilities written in clojure

Implemented so far:
 - cat
 - pwd
 - cd
 - exit

## Usage

lein uberjar
java -jar target\unix-clojure-0.1.0-SNAPSHOT-standalone.jar
A basic cmd like shell appears, type to enter command.

**lein run** doesn't work
